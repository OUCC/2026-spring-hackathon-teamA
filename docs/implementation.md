# FLOOR BREAKER — ゲーム仕様書

## 概要

テーマ「床」のリアルタイム対戦ゲーム。2人のプレイヤーが30x30のグリッドフィールドで、滑落ボムと炎ボムを駆使して戦う。スライムを倒してコインを稼ぎ、20秒ごとの強化フェーズでビルドを組み立てていく。時間経過とともにステージが縮小し、決着が迫る。

---

## 画面構成

- **分割画面**: プレイヤーごとに別々の画面（左: P1 / 右: P2）
- **フィールド**: 30x30タイル
- **カメラ**: 各プレイヤーを中心に追従、視界範囲は**縦横10マス**
- **HUD**: HP、コイン数、強化フェーズタイマー、取得済み強化一覧

---

## 操作

ゲームコントローラー前提。具体的なキーバインドは実装時に決定。

| 操作          | 説明                                                                                           |
| ------------- | ---------------------------------------------------------------------------------------------- |
| 移動（8方向） | スティック / 十字キー。**ボム飛行中も移動可能**（ボムの方向転換はしない）                |
| 滑落ボム      | **押している間ボムが飛行し続け**、ボタンを離すと着弾。最大飛行マスか障害物衝突で自動着弾 |
| 炎ボム        | 同上                                                                                           |

---

## プレイヤー

| パラメータ | 初期値      |
| ---------- | ----------- |
| 体力（HP） | 10          |
| 移動速度   | 1（基準値） |
| コイン     | 0           |

- 被弾時に短時間の無敵フレームあり

---

## ボム共通仕様

- 発射方向: 8方向（上下左右 + 斜め4方向）。最後に入力した方向が発射方向
- **飛行制御**: ボムボタンを押し続けている間飛行し、離すと着弾して効果発動
- **最大飛行距離**: デフォルト3マス（斜めも3マス）
- **衝突判定**: 壁・プレイヤー・スライムに衝突した時点で即着弾し効果発動
- **方向転換不可**: 飛行中にスティック入力してもボムの軌道は変わらない（移動はプレイヤーに適用）
- 着弾時の効果範囲: 着弾地点を中心に十字方向（上下左右）
- 効果距離: デフォルト1マス（着弾地点 + 十字各1マス = 計5マス）
- 放物線軌道で飛行する

---

## 滑落ボム

ヒットした床を一定時間崩落させる。

| パラメータ                       | 初期値                                                                               |
| -------------------------------- | ------------------------------------------------------------------------------------ |
| 最大飛行距離                     | 3マス                                                                                |
| 効果範囲                         | 十字1マス                                                                            |
| ダメージ（崩落時に上にいた場合） | 2                                                                                    |
| 崩落時間                         | 3秒                                                                                  |
| 崩落後の復帰時間                 | 5秒（崩落終了後から）                                                                |
| クールダウン                     | **4秒**（効果発動から次の発射まで）                                            |
| 飛行時ダメージ                   | なし                                                                                 |
| 壁貫通                           | **デフォルトで貫通する**（効果範囲が壁を無視して広がる。飛行自体は壁で止まる） |

### 崩落時のプレイヤー処理

崩落マスにプレイヤーがいた場合:

1. 2ダメージを受ける
2. **周囲3x3以内**の「崩落していない・プレイヤーやスライムがいない」マスに**強制移動**
3. 強制移動の演出時間は約**1秒**（この間操作不能）
4. **3x3以内に退避先がない場合**: 最も近い安全マスまで強制移動（BFS等で探索）

---

## 炎ボム

ヒットした床に火を発生させる。

| パラメータ     | 初期値                                    |
| -------------- | ----------------------------------------- |
| 最大飛行距離   | 3マス                                     |
| 効果範囲       | 十字1マス                                 |
| 接触時ダメージ | 1                                         |
| 継続ダメージ   | 1秒滞在ごとに1ダメージ                    |
| 炎の持続時間   | 3.5秒                                     |
| クールダウン   | **2秒**（効果発動から次の発射まで） |
| 飛行時ダメージ | なし                                      |
| 壁貫通         | しない                                    |

- 壁を破壊する

---

## 壁

| パラメータ       | 値                                 |
| ---------------- | ---------------------------------- |
| 初期配置率       | タイル総数の約20%                  |
| 配置アルゴリズム | 2パス方式（後述）                  |
| 再生             | なし（破壊されたら消滅）           |
| 破壊条件         | 滑落ボム・炎ボムの効果で破壊される |

### 壁配置アルゴリズム

1. **シードパス**: 全タイルの約8%をランダムに壁として配置
2. **成長パス**: シード壁に隣接する4方向のマスを40%の確率で壁に変換。全体が約20%になるまで繰り返す
3. **除外エリア**: P1・P2のスポーン地点を中心とした5x5範囲には壁を配置しない

ほぼランダムだが、シード+成長により自然な塊ができやすい。

### ボムと壁の関係

- **滑落ボムの飛行**: 壁に衝突すると着弾（飛行は貫通しない）
- **滑落ボムの効果範囲**: 壁を貫通して広がる（デフォルト）
- **炎ボムの飛行**: 壁に衝突すると着弾
- **炎ボムの効果範囲**: 壁で止まる（「延焼の壁貫通」強化で貫通可能に）
- いずれのボムも、効果範囲内の壁を**破壊する**

## スライム

フィールドに自然湧きする敵。倒すとコインまたは強化を得られる。

### 共通仕様

| パラメータ           | 値                                                               |
| -------------------- | ---------------------------------------------------------------- |
| 目標数               | 現存タイル数 x 3%（端数切り捨て）                                |
| スポーンチェック間隔 | 5秒ごと                                                          |
| スポーン位置条件     | 各プレイヤーから5マス以上離れた通行可能マス                      |
| HP                   | 1（ボムの効果に巻き込まれると即死）                              |
| 移動速度             | プレイヤーデフォルトの**1/2**                              |
| 索敵範囲             | **5マス以内**のプレイヤーに接近                            |
| 攻撃                 | **体当たり**（上下左右4方向の隣接マスから攻撃、1ダメージ） |
| 攻撃クールダウン     | **1秒**                                                    |

- 滑落ボム・炎ボムの効果範囲にいると死亡
- 崩落マスに巻き込まれても死亡

### バリエーション

| 種類             | 色   | ドロップ                                          | 出現比率     |
| ---------------- | ---- | ------------------------------------------------- | ------------ |
| ノーマルスライム | 水色 | コイン1枚                                         | **10** |
| 金色スライム     | 金色 | コイン**5枚**                               | **1**  |
| 赤色スライム     | 赤色 | 「無制限取得可能な強化」からランダムに1つ即時付与 | **1**  |

出現比率 10:1:1（ノーマル約83%、金色約8.3%、赤色約8.3%）

## 強化システム

### 強化フェーズ

- **20秒ごと**に発生
- **ゲーム一時停止**
- ステージ外周1列の崩落演出 -> 強化選択UIオーバーレイ（UIToolkit）
- ランダムに**3つの強化案**が提示される（ヴァンパイアサバイバーズ方式）
- **リロール**: コスト**1コイン**で3つの選択肢を引き直し（回数制限なし）
- コインが足りない強化はグレーアウト表示
- **制限時間10秒**。時間切れの場合は自動スキップ
- **両プレイヤーが選択またはスキップを完了**するとゲーム再開

### 強化一覧

#### 炎ボム強化

| 強化名                 | 効果                             | コスト | 重複            |
| ---------------------- | -------------------------------- | ------ | --------------- |
| 炎ボム飛距離増強       | 最大飛行距離+1マス               | 2      | 無制限          |
| 炎ボムフライ           | 効果範囲+1マス                   | 3      | 無制限          |
| 炎ボム威力増強         | 接触ダメージ+1                   | 3      | 無制限          |
| 炎ボム飛行時ダメージ   | 飛行中のボムに当たり判定追加     | 5      | 1回のみ（永続） |
| 炎ボム延焼時間増加     | 炎の持続時間+2秒                 | 2      | 無制限          |
| 延焼の壁貫通           | 炎の効果が壁を貫通するようになる | 6      | 1回のみ（永続） |
| 炎ボムクールダウン短縮 | クールダウン-0.3秒（下限0.5秒）  | 2      | 無制限          |

#### 滑落ボム強化

| 強化名                   | 効果                            | コスト | 重複            |
| ------------------------ | ------------------------------- | ------ | --------------- |
| 滑落ボム飛距離増強       | 最大飛行距離+1マス              | 3      | 無制限          |
| 滑落ボムフライ           | 効果範囲+1マス                  | 4      | 無制限          |
| 滑落ボム威力増強         | 崩落ダメージ+1                  | 4      | 無制限          |
| 滑落ボム飛行時ダメージ   | 飛行中のボムに当たり判定追加    | 6      | 1回のみ（永続） |
| 滑落ボム崩落時間増加     | 崩落持続時間+2秒                | 3      | 無制限          |
| 滑落ボムクールダウン短縮 | クールダウン-0.5秒（下限1.0秒） | 3      | 無制限          |

#### 汎用強化

| 強化名       | 効果                                      | コスト | 重複   | 備考                          |
| ------------ | ----------------------------------------- | ------ | ------ | ----------------------------- |
| 移動速度上昇 | 移動速度+0.2（上限2.0 = デフォルトの2倍） | 2      | 無制限 |                               |
| 体力回復     | HP3回復                                   | 2      | 無制限 | **HP5以下の時のみ出現** |

### 赤色スライムの強化ドロップ対象

赤色スライムを倒した際、以下の「無制限取得可能な強化」からランダムに1つが即時付与される:

炎ボム飛距離増強 / 炎ボムフライ / 炎ボム威力増強 / 炎ボム延焼時間増加 / 炎ボムクールダウン短縮 / 滑落ボム飛距離増強 / 滑落ボムフライ / 滑落ボム威力増強 / 滑落ボム崩落時間増加 / 滑落ボムクールダウン短縮 / 移動速度上昇

> **コスト設計方針**:
>
> - 滑落ボムの強化は炎ボムの同等強化より**1-2コイン高い**（壁貫通デフォルト + 移動封鎖 + 高ダメージのため）
> - 永続効果（飛行時ダメージ・壁貫通）は**高コスト**（5-6コイン）
> - 数値積み上げ系は中コストで無制限取得可能
> - クールダウン短縮には下限あり（炎0.5秒 / 滑落1.0秒）
> - 移動速度には上限あり（2.0 = デフォルトの2倍）

## ステージ縮小

- **20秒ごと**（強化フェーズと同時）に最外周1列が滑落して消滅
- 消滅マスは復帰しない（永久消滅）
- 消滅時にそのマスにいるプレイヤーは崩落処理（2ダメージ + 強制移動）
- 消滅時にそのマスにいるスライムは死亡（ドロップなし）

| フェーズ | 経過時間 | 残りフィールド   |
| -------- | -------- | ---------------- |
| 0        | 0:00     | 30x30（900マス） |
| 1        | 0:20     | 28x28（784マス） |
| 2        | 0:40     | 26x26（676マス） |
| 3        | 1:00     | 24x24（576マス） |
| 4        | 1:20     | 22x22（484マス） |
| 5        | 1:40     | 20x20（400マス） |
| ...      | ...      | ...              |
| 14       | 4:40     | 2x2（4マス）     |

## ゲームフロー

```
タイトル画面
  v START
フィールド生成（壁配置・スライム初期配置）
  v
ゲームループ開始
  +-- リアルタイム操作（移動・ボム発射）
  +-- スライム行動（5秒ごとにスポーンチェック）
  +-- 20秒経過
  |     +-- ゲーム一時停止
  |     +-- ステージ外周1列が崩落（永久消滅）の演出
  |     +-- 強化フェーズUI表示（UIToolkit オーバーレイ）
  |     +-- 両者選択完了 or 10秒経過
  |     +-- ゲーム再開
  +-- HP 0 -> ゲーム終了
  v
リザルト画面（勝者表示 -> REMATCH）
```

## 技術メモ

- **強化選択UI**: UIToolkit でオーバーレイ実装
- **入力**: ゲームコントローラー（Input System使用想定）
- **カメラ**: 分割画面は2カメラ構成、各プレイヤー追従
- **退避先探索**: 3x3内を優先探索、なければBFSで最近安全マスを検索
